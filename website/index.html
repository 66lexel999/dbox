D BOX is a free segmented download manager for Windows — multi-connection speed, pause & resume, crash recovery, YouTube & video capture, and one-click browser integration.

Segmented speed

Splits each file across many connections to saturate your line — far faster than a single stream.

⏯

Pause & resume

Stop any download and pick it up later. Survives crashes and reboots without losing progress.

🎬

Video & playlists

Grab YouTube videos, whole playlists, and Instagram reels via the built-in yt-dlp engine.

🔗

Browser capture

The companion extension catches downloads and hands them straight to D BOX — like IDM.

🔄

Auto-updates

D BOX updates itself in one click — no reinstalling, no losing your download list.

🎨

Themes & scale

Five themes, adjustable UI and text size, scheduling, and a Download-Later queue.

Installing

  1. Click Download for Windows above and run the installer.
  2. If Windows shows a “Windows protected your PC” screen, click More info → Run anyway (the app is safe; it's just not code-signed yet).
  3. D BOX opens and lives in your tray. New releases prompt you to update with one click.
Prefer no installer? Grab the portable DBox.exe and run it directly.
